O documento deve possuir:

* Um conteúdo de texto a ser exibido com título e ao menos duas sub-seções
* Uma sub-página com conteúdo

Folha de estilo separada
* Uma proposta de design consistente
* Utilizar todas as propriedades que vimos de configuração de BOX, texto e cores
* Possui ao menos uma imagem com texto alt
* Ter o conteúdo adaptado para a tela em que for lido, com tamanho de fonte adequado.
